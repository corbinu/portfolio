My Specialties

Node.js

I am a firm beliver in Asynchronous I/O. It makes highly conncurent and distributed applications easy to build. Node.js is very light weight and also adds the benefit of code sharing between the backend and frontend.

Titanium

When building a mobile app the first question people as is HTML5 or Native. Titanium sits in a middle ground, it provides a single JavaScript API that wraps the native SDKs for iOS, Android, Windows Phone and Blackberry. This means that the saved development time of a HTML5 app can be had at native performance.

Ceylon

There are a lot of new languages these days. I like Ceylon for two reasons. First its Union typing gives the flexibility of a dynamicly typed language with the code hinting and complile time checks of static typing. Second it compiles to both the JVM and the JSVM. This gives a massive amount of flexablity.

Ember.js

Building web apps is hard. I have found Ember.js to be a very effective tool. It does a great job of abstracting away templates using handlebars.js and centers the application around the URL. The Ember-Data service ties easily into a backend REST API.
